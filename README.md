# 🎾 Sistema de Reservas de Pickleball

Este proyecto es una aplicación web para gestionar la reserva de pistas de pickleball, desarrollada con **FastAPI** (backend) y **Angular** (frontend). Incluye integración con **Firestore** y envío de correos de confirmación mediante SMTP.

---

## 🚀 Tecnologías utilizadas

### Backend
- [FastAPI](https://fastapi.tiangolo.com/)
- Firebase Firestore
- Python 3.10+
- SMTP (Gmail)

### Frontend
- Angular
- TypeScript

---

## ⚙️ Instalación y ejecución

1. Clonar el repositorio

```bash
git clone https://github.com/emiliojg10/pickleball.git
cd pickleball

2. Configurar variables de entorno

SMTP_PASSWORD=tu_contraseña_de_aplicacion
PORT=8080

3. Instalar dependencias del backend

pip install -r requirements.txt

4. Ejecutar el backend

python main.py

5. Ejecutar el frontend (Angular)

cd pickleball-frontend
npm install
ng serve
